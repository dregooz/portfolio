MedifriendRx PWS

Kiosk Touchscreen App

About

MedifriendRx is owned by a company which builds and maintains pharmacies in medical clinics. Some clinics are too small to house a pharmacy, so this company developed a machine (PWS) that dispenses prescriptions to patients. Medical Assistants must use this built-in touchscreen app on the PWS to help patients retrieve prescriptions from the machine.

UI/UX

Visual design, user experience

Tools

Figma, Adobe Illustrator

Process

Sketching & Wireframing

· Brainstormed and sketched out interface ideas with client; client had a fairly specific idea of how the app should be laid out

· Medical Assistants would have to use the PWS touchscreen app to submit prescription dispense requests for processing and to walk patients through the dispense process; technicians would have to restock and unload inventory

· Client had developers build a very basic working prototype based off brainstorm sketches for testing

· Used this wireframe prototype to move forward with the final design

High Fidelity Mockups
& Visual Design

· Used wireframe prototype to create high-fidelity mockups; client would review them, I would iterate if needed

· Used the same color scheme and visual elements utilized on the MedifriendRx Web Portal for consistency
