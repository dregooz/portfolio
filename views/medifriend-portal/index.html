MedifriendRx Portal

Enterprise Web App

About

MedifriendRx is owned by a company which builds and maintains pharmacies in medical clinics. Some clinics are too small to physically house a pharmacy, so this company developed a dispensing machine that delivers prescriptions to patients. However, before a prescription can be dispensed, a pharmacist must remotely review and approve the transaction via this web portal.

UI/UX

Visual design, user experience

Tools

Figma, Adobe Illustrator

Process

Project Briefing

  • Client had a very specific idea of how the product should work and be laid out, and had an existing web app with basic functionality for testing
  • Pharmacist (user) would have to to approve, reject and see reviewed prescription dispense attempts, review machine activity and adjust inventory, access relevant personnel’s contact info, and more using the new web app
  • Took inventory of all features, functions and other elements in the existing web app

Sketching & Sitemap

  • Sketched out the app’s basic layout, then created a sitemap to keep everything in order at a glance

High Fidelity Mockups
& Visual Design

  • Client requested we jump directly into high-fidelity mockups; client would review them, iterations would be made if needed
  • Worked out the color scheme and visual elements while mocking up the first few screens, which was unique but interesting
  • Refined MedifriendRx’s blue accent color and remaining color scheme to work within a clean, grayscale interface
